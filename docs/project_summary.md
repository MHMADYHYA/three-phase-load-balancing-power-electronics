# Three-Phase Load Balancing and Power-Factor Correction Simulation

## Overview

Power electronics simulation project for three-phase load balancing, rectifier/DC-link/inverter modeling, RMS measurements, P/Q/S calculations, FFT/harmonic review, and power-factor improvement.

## Technical Highlights

- Three-phase AC input, rectifier, DC link, PWM inverter, and R-L loads.
- RMS voltage/current, active/reactive/apparent power, and PF calculations.
- FFT/harmonic analysis.
- Before/after load-balance comparison.
- Engineering conclusion about output filtering and modulation improvement.

## Tech Stack

Power Electronics, MATLAB, Simulink, PWM, FFT, Power Quality

## Results

- Base-case PF around 0.63 to 0.65.
- Balanced case reduces neutral current and improves phase-current symmetry.
- DC-link voltage around 977 to 980 V.
- Recommended next step: L/LCL output filter and SPWM/SVPWM optimization.
